# Proyecto final DEVF Modulo 1

Este proyecto es una página web diseñada para ser publicada mediante **GitHub Pages**. Incluye archivos HTML y otros recursos necesarios(Imagenes pero posteriormente quiero agregarle un archivo css) para mostrar el contenido de manera sencilla en línea.

## Características
- Página web estática.
- Estructura HTML básica y clara.
- Fácil de desplegar mediante GitHub Pages.

## Cómo desplegar el proyecto en GitHub Pages
1. Sube todos los archivos del proyecto a un repositorio en GitHub.
2. Ve a **Settings** del repositorio.
3. En el menú lateral selecciona **Pages**.
4. En "Source", elige la rama donde está tu proyecto (por lo general `main`).
5. Selecciona la carpeta `/root` (o `/docs` si usas esa estructura).
6. Guarda los cambios.

GitHub generará un enlace parecido a:
```
https://estorealsstudent.github.io/Proyecto_Final_DF-/
```
